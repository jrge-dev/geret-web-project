# Sistema de Gestión de Infraestructura de Telecomunicaciones

Aplicación web desarrollada en **PHP** para prueba técnica.
El objetivo principal fue contruir una solución web funcional que gestione el estado y registro de infraestructura de telecomunicaciones bajo requisitos de seguridad y administración especificos.

## Requisitos Resueltos

- **Gestión de sesiones**: Implementación de rutas protegidas mediante el manejo de sesiones nativas de PHP y validación de usuarios en base de datos.
- **Operaciones CRUD**: Panel administrativo para la creación. listado y actualización de estado de antenas.
- **Persistencia de datos**: Diseño y consulta de base de datos SQL para asegurar la integridad de la información.

---

_Este proyecto fue realizado bajo los lineamientos de una prueba técnica, priorizando el cumplimiento de las funcionalidades requeridas._
